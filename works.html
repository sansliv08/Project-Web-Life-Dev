Kanban Board

A kanban board is an agile project management tool designed to help visualize work, limit work-in-progress, and maximize efficiency. It can help both agile and DevOps teams establish order in their daily work. It is possible to drang and drop each item to the different columns, create new items, update and delete existing items. It also allows saving in localStorage. Try in Google Chrome.

Infinitive Scroll

This is a simple training app based on HTML, CSS and Javascript. It allows fetch the data in JSON format from the Unsplash Api and insert the photos into the DOM on initial loading. When the user scrolls close to the bottom of the page the app makes another GET request to the API and inserts photos into the DOM (lazy loading).
