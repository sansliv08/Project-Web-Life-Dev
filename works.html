Media Player

Kanban Board

A kanban board is an agile project management tool designed to help visualize work, limit work-in-progress, and maximize efficiency. It can help both agile and DevOps teams establish order in their daily work.

It is possible to drang and drop each item for different columns, create new items, update and delete existing items. It also allows saving in localStorage.

Infinitive Scroll

This is a simple training app based on HTML, CSS and Javascript. It allows fetch the data in JSON format from the Unsplash Api and insert the photos into the DOM on initial loading.

When the user scrolls close to the bottom of the page the app makes another GET request to the API and inserts photos into the DOM (lazy loading).

Pong

This project uses Canvas HTML elements to create a classic pong game. Users play against the computer. The player who reaches 7 points, is the winner.

Paint

This project is a very basic paint app, that uses Canvas API. It allows draw on canvas (brush), change brush color, change canvas background color (brucket), erase draw element (eraser), change brush/eraser thickness (slider), clear canvas, save/load/clear localStorage and save as an image file.

Custom Countdown

With javascript, this project allows the user to enter a date and the respective event or task name and it will count down. It stores countdown information in local storage, so users are free to refresh page or close the browser without losing data.

Form validation
